# AI Features Guide — RTBPF CMS
> คู่มือการใช้งานฟีเจอร์ AI ทั้งหมดในระบบ

---

## 1. 🎨 AI Image Generation (Kie.ai)

### ตำแหน่งในระบบ
- **Admin** → Articles → Create/Edit → Sidebar ขวา → **AI IMAGE**

### วิธีใช้งาน
1. สร้าง/แก้ไขบทความ → กรอกหัวข้อข่าว
2. เลื่อนลงมาที่ **AI IMAGE** section ด้านขวา
3. เลือก **Style** (สไตล์ภาพ):
   - `ภาพข่าว` — ภาพถ่ายข่าวสมจริง
   - `อินโฟกราฟิก` — กราฟิกข้อมูล
   - `Illustration` — ภาพวาดประกอบ
   - `อะนิเมะ` — สไตล์อนิเมะ
   - `Digital Art` — ศิลปะดิจิทัล
   - `วาดมือ` — สเก็ตช์ลายเส้น
   - `Cinematic` — ภาพยนตร์
   - `Minimal` — มินิมัล
4. เลือก **Aspect Ratio**: 16:9 (ปกข่าว), 1:1 (โซเชียล), 4:3 (บทความ)
5. (Optional) พิมพ์คำสั่งเพิ่มเติม เช่น "สีโทนทอง, ธงชาติไทย"
6. กดปุ่ม **"✨ สร้างภาพ AI"**
7. รอ ~30-60 วินาที → ภาพจะปรากฏ
8. กด **"ใช้เป็นภาพปก"** → ภาพจะไปเป็น Featured Image

### Watermark "Gen From AI"
- ภาพจาก AI ทุกภาพจะมี badge **"★ GEN FROM AI"** ที่มุมล่างซ้ายบนหน้าเว็บสาธารณะ
- Badge แสดงอัตโนมัติ — ไม่สามารถปิดได้ (100% compliance)
- ตรวจจับจาก path `ai-generated/` ใน URL ภาพ

### ค่าใช้จ่าย
- ~$0.003 ต่อภาพ (~0.10 บาท)

## 2. 🔊 Text-to-Speech (TTS)

### ตำแหน่งในระบบ
- **หน้าสาธารณะ** → อ่านข่าว → ปุ่ม **"🔊 ฟังบทความ"**

### วิธีใช้งาน
1. เปิดบทความที่เผยแพร่แล้ว (Published) บนเว็บ
2. กดปุ่ม **"🔊 ฟังบทความ"** ที่ด้านบนเนื้อหา
3. รอ 2-5 วินาที → Audio Player จะปรากฏที่ก้นหน้าจอ
4. ควบคุมได้: Play/Pause, ความเร็ว (1x), ปิดเสียง, ปิด Player

### เทคนิค
- ใช้ **Google Cloud TTS** เป็นหลัก
- Fallback เป็น **Browser Web Speech API** ถ้า API ไม่พร้อม
- เสียงจะถูก cache ใน Supabase Storage — ครั้งต่อไปจะเร็วขึ้น

### ค่าใช้จ่าย
- ~$0.002 ต่อบทความ (~0.07 บาท) สำหรับบทความ ~500 ตัวอักษร

## 3. 🌐 Multi-Language Translation

### ตำแหน่งในระบบ
- **Admin** → Articles → Edit → Sidebar ขวา → **AI MULTI-LANGUAGE**

### ภาษาที่รองรับ
| Code | ภาษา | ธง |
|------|------|-----|
| th | ไทย | 🇹🇭 (ต้นฉบับ) |
| en | English | 🇬🇧 |
| ko | 한국어 | 🇰🇷 |
| ja | 日本語 | 🇯🇵 |
| zh | 简体中文 | 🇨🇳 |
| fr | Français | 🇫🇷 |
| de | Deutsch | 🇩🇪 |
| es | Español | 🇪🇸 |

### วิธีใช้งาน
1. เปิดบทความที่ต้องการแปล → หน้า Edit
2. ดูที่ sidebar ขวา → **AI MULTI-LANGUAGE**
3. กด **"Translate All"** เพื่อแปลทุกภาษาพร้อมกัน
4. หรือกด **"TRANSLATE"** ทีละภาษา
5. รอ 5-15 วินาทีต่อภาษา
6. เมื่อแปลเสร็จ → badge จะเปลี่ยนเป็น "Translated"

### เทคนิค
- ใช้ **Gemini 2.0 Flash** ผ่าน Google AI Studio API
- แปลทั้ง title, excerpt, และ content (TipTap JSON format)
- บันทึกลง table `article_translations`
- URL: `/th/articles/xxx`, `/en/articles/xxx`, `/ja/articles/xxx` ฯลฯ

### ค่าใช้จ่าย
- ~$0.00002 ต่อภาษา → 7 ภาษา = ~$0.0001 (~0.01 บาท)
